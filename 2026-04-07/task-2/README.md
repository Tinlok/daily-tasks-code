# Unity 游戏基础UI系统

这是一个Unity游戏的基础UI系统，实现了主菜单、暂停菜单和设置菜单功能。

## 文件结构

- `MainMenu.cs` - 主菜单控制器，处理开始游戏、设置和退出功能
- `PauseMenu.cs` - 暂停菜单控制器，处理游戏暂停时的各种操作
- `SettingsMenu.cs` - 设置菜单控制器，处理音量、画质和控制设置
- `UIManager.cs` - UI管理器，负责UI面板的切换和管理

## 功能特性

### 1. 主菜单 (MainMenu)
- 开始游戏按钮
- 设置按钮（跳转到设置菜单）
- 退出游戏按钮
- 淡入动画效果

### 2. 暂停菜单 (PauseMenu)
- 继续游戏按钮
- 重新开始按钮
- 设置按钮（跳转到设置菜单）
- 退出到主菜单按钮
- 退出游戏按钮
- ESC键支持（暂停/恢复）
- 淡入淡出动画效果

### 3. 设置菜单 (SettingsMenu)
- **音量设置**：
  - 主音量滑块
  - 音乐音量滑块
  - 音效音量滑块
  - 静音开关
- **画质设置**：
  - 质量等级下拉框
  - 全屏开关
  - 垂直同步开关
- **控制设置**：
  - 控制方案下拉框
  - 重新绑定按钮

### 4. UI管理器 (UIManager)
- 单例模式设计
- 场景自适应UI显示
- 面板切换动画
- 跨场景UI保持

## 使用方法

### 1. 场景设置
在Unity编辑器中：
1. 创建Canvas对象
2. 在Canvas下创建各个UI面板（主菜单、暂停菜单、设置菜单、游戏HUD）
3. 为每个面板添加对应的脚本组件
4. 在UIManager中引用这些面板

### 2. 组件绑定
在每个UI脚本中，需要将相应的UI元素拖拽到Inspector面板中：
- Button对象
- Slider对象
- Toggle对象
- Dropdown对象
- CanvasGroup对象

### 3. 场景切换
- 创建"MainMenu"和"Game"场景
- 在相应场景中设置UIManager对象
- 根据场景名称自动显示相应的UI

## 配置要求

- Unity 2020.3 或更高版本
- Unity UI系统（Canvas, Button, Slider, Toggle, Dropdown等）
- 支持C# 9.0语法

## 设置保存

所有设置会自动保存到PlayerPrefs中：
- 音量设置
- 画质设置
- 全屏设置
- 控制方案设置

## 动画效果

系统包含以下动画效果：
- 主菜单淡入
- 暂停菜单淡入淡出
- 设置菜单切换动画
- 所有动画时间可通过参数调整

## 扩展建议

1. **添加更多UI元素**：如加载界面、游戏结束界面等
2. **皮肤系统**：为UI添加主题切换功能
3. **本地化**：添加多语言支持
4. **输入重绑定**：实现完整的按键重新绑定功能
5. **音效系统**：为UI操作添加音效反馈

## 注意事项

1. 确保所有UI面板都有CanvasGroup组件用于动画
2. 在不同场景中切换时，UIManager会自动处理UI显示
3. 设置保存使用PlayerPrefs，适合简单设置需求
4. 复杂设置建议使用ScriptableObject或文件系统
